# u.taiy.me

taiy専用URL短縮サービスのソースコードです。 <https://u.taiy.me/>

## 仕様

- 言語
  - TypeScript <https://www.typescriptlang.org/>
- フレームワーク
  - Hono <https://hono.dev/>
- Linter
  - ESLint (Flat Config) <https://eslint.org/>

## 開発

前提: Corepackが有効であること

```sh
# 依存関係のインストール
pnpm i

# 本番環境にデプロイ
pnpm run deploy
```
